To run this program, use the following commands:

sudo java -jar 2AA4_Assignment_2.jar

Alternatively: 
	1. Check the permissions on the bash file "run" [chmod u+x run]
	2. Make sure the bash script is in the same folder as the jar file.
	3. Run the bash script. [./run]

Note: When saving game, the data will be saved in the root directory of your computer. Be sure to delete it after.
Note: When you do not enter a filename to load or save, the default is gamedata.txt
